# Dore Backend

## Project setup

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## New Module/Controller/Service

```bash
# 예: users 모듈 생성
nest generate module users
# 컨트롤러
nest generate controller users
# 서비스
nest generate service users
```

src/users/ 아래에 module, controller, service 파일 생성
